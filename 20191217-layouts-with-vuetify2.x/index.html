<v-app>




  • 他のVuetifyのコンポーネントの親になるもので、アプリケーション内で1つ用意する

Application components




  • app propを持つコンポーネント
  • 該当するコンポーネント
    • v-system-bar
    • v-app-bar
    • v-navigation-drawer
    • v-footer
    • v-bottom-navigation
  • 配置する場所が決められている (e.g. v-system-barはアプリの最上部)

<v-content>




  • Application componentsに応じて動的にサイズが決まるコンポーネント
  • <main>の中に入る要素をこの中に入れるとよい (e.g. Vue Routerの<router-view>)
